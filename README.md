# Papyrus

Paper generation based on templates. Web-based printing using print-client.

## Usage

How to use my plugin.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'papyrus'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install papyrus
```
